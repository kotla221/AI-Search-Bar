# AI Search Tab

A Chrome extension that replaces the new tab page with one clean search box and quick engine choices for Google, ChatGPT, Claude Code, Gemini AI Mode, Perplexity, Grok, Reddit, X, and Amazon.

## Features

- Minimal tech-style homepage with dark and light modes.
- Remembers your selected engine and lets you set the selected engine as default.
- Enter searches in the current tab.
- Shift Enter opens the search in a new tab.

## Load In Chrome

1. Open `chrome://extensions`.
2. Turn on **Developer mode**.
3. Click **Load unpacked**.
4. Select this folder: `folder_path`.
